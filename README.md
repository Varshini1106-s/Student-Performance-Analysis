# Student Performance Analysis

This project analyzes the Student Performance dataset using Python and Google Colab.

## Tools Used
- Python
- Pandas
- Matplotlib

## Tasks Performed
- Data Loading
- Data Cleaning
- Missing Value Check
- Duplicate Removal
- Statistical Analysis
- Data Visualization

## Visualizations
- Histogram of Final Grades
- Scatter Plot: Study Time vs Final Grade
- Bar Chart: Average Final Grade by Gender

## Results
- Average Final Grade: 10.42
- Students Scoring Above 15: 40
- Correlation between Study Time and Grade: 0.0978
- Male students performed slightly better on average.
